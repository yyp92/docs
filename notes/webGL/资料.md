## threejs

### 本地搭建threejs项目

- github中下载dev分支代码

```bash
# 克隆项目代码
git clone -b dev git@github.com:mrdoob/three.js.git
# 或者直接下载压缩包

# 安装依赖包
npm install
yarn
pnpm install

# 启动项目
npm run start
yarn start
pnpm start
```

- 在本地可以方便的查看样例，根据样例的url去找相应的代码
